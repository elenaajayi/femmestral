

# Femstral – Health Fact-Check Extension UI

## Page 1: Reddit Thread Mockup with Inline Fact-Check Panels
- Recreate a simplified Reddit thread view (post about perimenopause joint pain, based on the reference screenshots)
- Below each comment reply, embed a **Femstral Fact-Check panel** with:
  - Femstral logo icon
  - Color-coded badge (green/yellow/red) with status text
  - 2–3 sentence evidence summary
  - Confidence score (e.g., "82% confidence")
  - Citation chips (Mayo Clinic, NIH, ACOG)
  - "View Trusted Resource" link
- Show all three states across different comments: Supported, Partially Supported, Not Evidence-Based

## Page 2: Text Selection Pop-Up
- Same Reddit thread view
- Show highlighted text within a comment
- Floating **"Femstral Insight"** micro-panel (max 300px) near the selection with:
  - Fact-check badge
  - One-paragraph explanation
  - "Learn More" link
  - Expandable "Why this matters" accordion
  - Subtle shadow, rounded corners

## Page 3: Component Library
- All badge variants (green, yellow, red) with labels
- Inline panel component in all 3 states
- Floating tooltip card component
- Button styles ("View Trusted Resource", "Learn More")
- Typography samples
- Color palette swatches (soft purple, white, muted teal)

## Design System
- Colors: soft purple (#7C5CFC), muted teal (#4ECDC4), white backgrounds, semantic green/yellow/red
- Font: Inter (sans-serif)
- Light mode only
- Accessible contrast ratios
- Clean health-tech aesthetic with soft borders and rounded corners
